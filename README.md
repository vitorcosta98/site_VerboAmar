# Site Interno para a ONG Verbo Amar

A ONG Verbo Amar solicitou a criação de um site para auxílio da rotina administrativa.

As funções esperadas do projeto.
- 
* Registro de faltas dos alunos nas aulas;
* Cadastro e Exclusão de alunos;
* Cadastro e Exclusão de Professores;
* Criação e Exclusão de Cursos;
* Criação e Exclusão de Turmas;
